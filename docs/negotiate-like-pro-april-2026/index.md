Tech salary negotiation is not just about asking for a higher number. It's about understanding the company's budget, the market rate for the position, and the value you bring to the table. According to a survey by LinkedIn, 68% of employers consider salary negotiations to be a normal part of the hiring process. To negotiate effectively, you need to do your research and come prepared with data. For example, you can use online tools such as Payscale or Indeed to determine the average salary for your position in your area. You can also use scripts like the following to guide your negotiation:
```python
def negotiate_salary(desired_salary, company_budget):
  if desired_salary <= company_budget:
    return desired_salary
  else:
    return company_budget * 0.9
```
This script takes into account the company's budget and your desired salary, and returns a compromise that is 10% below your desired salary.

## Advanced Configuration and Edge Cases
When it comes to advanced configuration and edge cases, there are several factors to consider. For example, what if you're negotiating a salary with a company that has a complex compensation structure, including bonuses, stock options, and benefits? In this case, you may need to use a more advanced script that takes into account these additional factors. For example:
```python
def negotiate_complex_salary(desired_salary, company_budget, bonuses, stock_options, benefits):
  total_compensation = desired_salary + bonuses + stock_options + benefits
  if total_compensation <= company_budget:
    return total_compensation
  else:
    return company_budget * 0.9
```
This script takes into account the company's budget, your desired salary, and the additional compensation factors, and returns a compromise that is 10% below your desired total compensation. Another edge case to consider is what if you're negotiating a salary with a company that has a strong culture of negotiation, where the employer expects you to negotiate aggressively? In this case, you may need to use a more aggressive script that pushes the boundaries of what is possible. For example:
```python
def negotiate_aggressive_salary(desired_salary, company_budget):
  if desired_salary <= company_budget:
    return desired_salary * 1.1
  else:
    return company_budget * 0.95
```
This script takes into account the company's budget and your desired salary, and returns a compromise that is 10% above your desired salary, or 5% below the company's budget, whichever is higher.
